LatestProject

Enhancing Local News Web App with Advanced Features

This project is a smart web application that allows users to submit local news, which is then verified using a machine learning model trained on the WELFake dataset. Verified news is displayed on an interactive map, and sentiment analysis is applied using TextBlob to detect the tone of the content. The app supports user authentication and provides a multilingual interface, making local news more accessible, trustworthy, and user-friendly.

SmartInspector AI

SmartInspector AI is a home inspection tool that leverages YOLOv8 for real-time image-based damage detection. Users can upload photos of household structures or appliances, and the system identifies damage types and estimates their severity. The application automatically generates a detailed PDF report that includes annotated images, repair cost estimates, and severity analysis, helping inspectors or homeowners make informed decisions quickly.

Expense Report Automation Using RPA

This project automates the end-to-end process of submitting and processing expense reports using UiPath RPA. It captures data from invoices and receipts, validates them against predefined rules, and enters the details into the financial system. The bot also handles approvals and email notifications. This automation reduces manual errors, improves efficiency, and saves significant time in business expense workflows.
